# jeremylasman.com

Personal portfolio and writing site for Jeremy Lasman — speaker, creative entrepreneur, and founder.

**Live site:** [jeremylasman.com](https://jeremylasman.com)

## Site Structure

| Path | Description |
|------|-------------|
| `/` | Homepage |
| `/writing/` | Essays and long-form writing |
| `/speaking/` | Speaking engagements and talks |
| `/press/` | Press mentions and media |
| `/now/` | Now page — what I'm currently focused on |
| `/work-with-me/` | Services and collaboration |
| `/links/` | Link-in-bio page |

## Tech Stack

- Static HTML, CSS, and vanilla JavaScript
- Hosted on **GitHub Pages** with a custom domain
- Analytics via Google Analytics

## Local Development

No build step required. Serve the root directory with any static server:

```bash
# Using Python
python3 -m http.server

# Using Node
npx serve
```

Then open `http://localhost:8000` (or `:3000` for serve).
